Lower Difficulty

  • Basic CRUD Operations: Product management, user registration, and authentication flows
  • UI Components: Navigation, forms, and responsive design implementation
  • Route Management: Setting up Express.js routes and middleware

Our application supports the following server-side routes:

Authentication Routes

/register - User registration POST
/login - User authentication POST
/logout - User logout POST

Store & Product Routes

/products - Get all products GET
/add-to-cart - Add product to cart POST

Cart Management Routes

/cart - Get user's cart GET
/remove-from-cart - Remove item from cart POST
/remove-all-from-cart - Remove all instances of a product POST

Purchase & Checkout Routes

/checkout - Process checkout POST
/my-items - Get user's purchase history GET

Admin Routes

/admin-activity - Get user activity logs GET
/admin-products - Get all products for admin GET
/admin-add-product - Add new product POST
/admin-remove-product - Remove product (POST method) POST
/admin-products/:id - Remove product (DELETE method) DELETE
